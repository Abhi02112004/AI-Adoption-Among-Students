 AI Adoption Among Students

A Python-based data analysis project that explores how students use Artificial Intelligence (AI) tools in their academic journey. This project focuses on data cleaning, data transformation, exploratory data analysis (EDA), and insight generation using a real-world student survey dataset.

Project Overview

The rapid adoption of AI tools such as ChatGPT, Gemini, Copilot, and Claude is transforming the way students learn, complete assignments, and develop technical skills. This project analyzes student responses to understand:

* AI tool adoption trends
* Daily AI usage patterns
* Academic impact of AI tools
* Student trust in AI-generated responses
* Preferred AI tools
* State-wise and stream-wise usage patterns
* Device and internet accessibility trends
* Willingness to pay for AI services

 Dataset Features

The dataset contains the following attributes:

* Student Name
* College Name
* Stream
* Year of Study
* AI Tools Used
* Daily Usage Hours
* Use Cases
* Trust in AI Tools
* Impact on Grades
* Professor Approval of AI Usage
* Preferred AI Tool
* Awareness Level
* Willingness to Pay for Access
* State
* Device Used
* Internet Access
